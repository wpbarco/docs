To install the LangChain package:



    <CodeGroup>
    ```bash npm
    npm install langchain@next @langchain/core@next
    ```

    ```bash pnpm
    pnpm add langchain@next @langchain/core@next
    ```

    ```bash yarn
    yarn add langchain@next @langchain/core@next
    ```

    ```bash bun
    bun add langchain@next @langchain/core@next
    ```
    </CodeGroup>


LangChain provides integrations to hundreds of LLMs and thousands of other integrations. These live in independent provider packages. For example:



<CodeGroup>
    ```bash npm
    # Installing the OpenAI integration
    npm install @langchain/openai@next
    # Installing the Anthropic integration
    npm install @langchain/anthropic@next
    ```

    ```bash pnpm
    # Installing the OpenAI integration
    pnpm install @langchain/openai@next
    # Installing the Anthropic integration
    pnpm install @langchain/anthropic@next
    ```

    ```bash yarn
    # Installing the OpenAI integration
    yarn add @langchain/openai@next
    # Installing the Anthropic integration
    yarn add @langchain/anthropic@next
    ```

    ```bash bun
    # Installing the OpenAI integration
    bun add @langchain/openai@next
    # Installing the Anthropic integration
    bun add @langchain/anthropic@next
    ```
</CodeGroup>


<Tip>
See the [Integrations tab](/oss/javascript/integrations/providers/overview) for a full list of available integrations.
</Tip>
