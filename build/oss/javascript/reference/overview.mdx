These pages detail the core interfaces you will use when building with LangGraph. Each section covers a different part of the ecosystem.

## LangGraph

The core APIs for the LangGraph open source library.



- [Graphs](https://langchain-ai.github.io/langgraphjs/reference/classes/langgraph.Graph.html): Main graph abstraction and usage.
- [Functional API](https://langchain-ai.github.io/langgraphjs/reference/modules/langgraph.html): Functional programming interface for graphs.
- [Pregel](https://langchain-ai.github.io/langgraphjs/reference/classes/langgraph.Pregel.html): Pregel-inspired computation model.
- [Checkpointers](https://langchain-ai.github.io/langgraphjs/reference/modules/checkpoint.html): Saving and restoring graph state.
- [MemorySaver](https://langchain-ai.github.io/langgraphjs/reference/classes/checkpoint.MemorySaver.html): Storage backends and options.
- [Errors](https://langchain-ai.github.io/langgraphjs/reference/variables/checkpoint.ERROR.html): Error types and handling.
- [Channels](https://langchain-ai.github.io/langgraphjs/reference/classes/langgraph_pregel.Channel.html): Message passing and channels.


## Prebuilt components

Higher-level abstractions for common workflows, agents, and other patterns.



- [Agents](https://langchain-ai.github.io/langgraph/reference/agents/): Built-in agent patterns.
- [Supervisor](https://langchain-ai.github.io/langgraph/reference/supervisor/): Orchestration and delegation.
- [Swarm](https://langchain-ai.github.io/langgraph/reference/swarm/): Multi-agent collaboration.
