>[MongoDB](https://www.mongodb.com/) is a NoSQL, document-oriented
> database that supports JSON-like documents with a dynamic schema.

**NOTE:**
- See other `MongoDB` integrations on the [MongoDB Atlas page](/oss/python/integrations/providers/mongodb_atlas).

## Installation and Setup

Install the Python package:

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
